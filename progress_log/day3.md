## What I Learned
- `selectbox()` is used to choose one option from a list
- `slider()` is used to select a number from a range
- `text_area()` allows users to write larger text/reflections
- Dictionaries store related information together using key-value pairs
- `session_state` acts like temporary memory for the app
- Streamlit reruns the script after every interaction

---

## Main Concept Learned Today
Apps become more intelligent when they can collect and remember human behavior data.
